# PDF Studio ✨

**Xoay trang • Ký tên • Comment • Zoom • Ghép PDF** — Tất cả chạy **hoàn toàn trên trình duyệt**, không cần cài đặt, không upload lên server.

![PDF Studio](https://img.shields.io/badge/PDF%20Studio-Modern%20PDF%20Editor-orange?style=for-the-badge)

### 🌟 Tại sao nên dùng PDF Studio?

- **Hoàn toàn offline** — Toàn bộ xử lý diễn ra ngay trên máy bạn
- **Giao diện đẹp, mượt mà**, thiết kế tối giản theo phong cách giấy ấm
- **Hỗ trợ đa ngôn ngữ**: Tiếng Việt • English • ภาษาไทย
- **Chữ ký chuyên nghiệp** với ngày giờ + **QR Code xác thực** (nhúng thời gian + IP)
- **Comment linh hoạt** — kéo thả, đổi màu, đổi kích thước, chỉnh sửa trực tiếp
- **Xoay trang hàng loạt** hoặc từng trang
- **Xuất trang chọn lọc** (có thể chọn lẻ, chẵn, dải trang...)
- **Ghép nhiều file PDF** thành một (kéo thả sắp xếp thứ tự, hỗ trợ chọn dải trang)

### ✨ Tính năng nổi bật

| Tính năng              | Mô tả |
|------------------------|------|
| **📄 Xoay trang**      | Xoay từng trang hoặc xoay tất cả (+90°, 180°, 270°) |
| **✍️ Chữ ký**          | Vẽ tay / Gõ chữ / Upload ảnh • Thêm ngày giờ • QR Code bảo mật |
| **💬 Comment**         | Thêm chú thích màu sắc, kéo thả, thay đổi kích thước |
| **🔍 Zoom mượt**       | Zoom tự do, vừa trang, hỗ trợ pinch zoom trên mobile |
| **📑 Xuất trang chọn** | Chọn trang linh hoạt (tất cả / lẻ / chẵn / dải / thủ công) |
| **🔗 Ghép PDF**        | Ghép nhiều file, sắp xếp thứ tự, chọn dải trang từng file |
| **⬇️ Tải PDF**         | Xuất file PDF đã chỉnh sửa chất lượng cao |

### 🚀 Cách sử dụng

1. **Mở file PDF**  
   - Kéo thả file vào khu vực giữa màn hình  
   - Hoặc nhấn **"Chọn file PDF"**

2. **Chỉnh sửa**
   - Xoay trang bằng nút trên thanh công cụ hoặc sidebar
   - Tạo chữ ký → Chọn **"✍ Đặt chữ ký"** → Click vào trang
   - Thêm comment → Chọn **"💬 Thêm comment"** → Click vào trang
   - Chọn trang muốn xuất (nếu cần)

3. **Hoàn tất**
   - Nhấn **⬇ Tải xuống PDF** để lưu file đã chỉnh sửa
   - Hoặc dùng **"📑 Xuất trang"** / **"🔗 Ghép PDF"**

---

### 🛠️ Công nghệ sử dụng

- **pdf-lib** — Xử lý và tạo PDF
- **pdf.js** — Render trang PDF mượt mà
- **Canvas API** — Vẽ chữ ký, comment, thumbnail
- **QRCode.js** — Tạo QR Code xác thực
- **Hoàn toàn chạy bằng JavaScript** (không backend)
